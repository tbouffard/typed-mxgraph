Class mxSwimlane

Extends mxShape to implement a swimlane shape. This shape is registered under mxConstants.SHAPE_SWIMLANE in mxCellRenderer. Use the {@link mxConstants.STYLE_STYLE_STARTSIZE} to define the size of the title region, mxConstants.STYLE_SWIMLANE_FILLCOLOR for the content area fill, mxConstants.STYLE_SEPARATORCOLOR to draw an additional vertical separator and mxConstants.STYLE_SWIMLANE_LINE to hide the line between the title region and the content area. The mxConstants.STYLE_HORIZONTAL affects the orientation of this shape, not only its label.
